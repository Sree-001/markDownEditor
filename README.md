## Steps to run the code. 

 > 1)  `git clone   https://github.com/Sree-001/markDownEditor.git markDown` 
  
 > 2)  `cd markDown && npm install ` 

 > 3)  `npm start`  

 > 4) Navigate to  [http:localhost:3000](http:localhost:3000)  to view the App
